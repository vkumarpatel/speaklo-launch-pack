# Speaklo — Launch Pack

Speaklo is a lightweight demo / launch pack for an AI-powered English speaking app.
This repo includes:
- `index.html` — demo (Lottie animated avatar + mic + TTS + mouth animation)
- `speaklo_logo.svg` — logo SVG
- `microcopy_speaklo.txt` — English + Hindi microcopy
- `Instructions.txt` — hosting and Play Store tips

## Quick start (local)
1. Download this repo (or clone it).
2. Download a free Lottie JSON from LottieFiles (search: `character celebrate confetti lottie`) and save it as `speaklo_tala.json` in the same folder as `index.html`.
3. Run a simple static server (recommended) and open in Chrome:
   ```bash
   npm install -g serve
   serve .
