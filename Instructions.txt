Local test:
1. Put these files in a folder: index.html, speaklo_logo.svg, microcopy_speaklo.txt.
2. Download a free Lottie JSON from LottieFiles (search: "character celebrate confetti lottie") and save as speaklo_tala.json in the same folder.
3. Open index.html in Chrome (desktop) or use a local server (e.g., `npx serve .`) for best microphone permissions.

Host on Vercel:
1. Create a GitHub repo and push these files.
2. Connect the repo to Vercel and deploy (free plan).
3. Share yourproject.vercel.app and test on Android Chrome. Use 'Add to Home screen' to install PWA.
